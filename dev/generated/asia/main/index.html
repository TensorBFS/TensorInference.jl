using TensorInference

instance = read_instance(pkgdir(TensorInference, "examples", "asia", "asia.uai"))

Since the evidence variable may change the contraction order, we re-compute the tensor network.

tnet = TensorNetworkModel(instance)

Get not only the maximum log-probability, but also the most probable conifguration In the most probable configuration, the most probable one is the patient smoke (3) and has lung cancer (4)

logp, cfg = most_probable_config(tnet)
(-3.65222179200233, [1, 1, 0, 0, 0, 0, 0, 0])

Get the maximum log-probabilities (MMAP) To get the probability of lung cancer, we need to marginalize out other variables.

mmap = MMAPModel(instance; queryvars=[4,7])
MMAPModel{Int64, Array{Float64}}
variables: 4, 7 (evidence → 0)
query variables: [[1, 2, 6, 5, 3, 8]]
contraction time = 2^6.044, space = 2^2.0, read-write = 2^7.044

We get the most probable configurations on [4, 7]

most_probable_config(mmap)
(-2.8754627318176693, [1, 0])

The total probability of having lung cancer is roughly half.

log_probability(mmap, [1, 0]), log_probability(mmap, [0, 0])
(-2.8754627318176693, -2.920624801067186)
